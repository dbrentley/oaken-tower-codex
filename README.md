# Oaken Tower Codex

Static codex of every item, potion, slot buff, and encounter in [Oaken Tower](https://store.steampowered.com/app/3400960/Oaken_Tower/).

Generated from the game's `data.win` binary (no external API required). All sprite art and stat values are extracted directly from the local game install.

**Open `index.html` in any browser**, or visit [https://dbrentley.github.io/oaken-tower-codex/](https://dbrentley.github.io/oaken-tower-codex/).

Source pipeline: https://github.com/dbrentley/oaken-tower-codex
(See `extract/` in the workspace repo for the extractors.)
